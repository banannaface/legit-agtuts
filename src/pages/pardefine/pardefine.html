Defining Parabola

Parabola is the curve formed by the set of points in a plane that are all equally distant from both a given line (directrix) and a given point (focus) which is not on the directrix.

Standard Equation of Parabola (vertex at h,k)

 There are also parabolas whose vertex does NOT lie at the origin. Notice that we still have almost the same formula as the one with the vertex at origin but this time we subtracted the h from x and k from y. Hence when (h,k) is (0,0) or at origin, we still have the same formula.

General Equation of Parabola

Ax2 + Cx + Dy + E = 0

(A and C are nonzero)

or
By2 + Cx + Dy + E = 0

(B and C are nonzero)

If the equation is given in general form, we can determine the standard form by completing the square in both variables.
Example 1: y2 - 5x + 12y = -16

y2 + 12y = 5x - 16 y2 + 12y + 36 = 5x - 16 + 36 (y + 6)2 = 5x + 20 (y + 6)2 = 5 (x + 4)
